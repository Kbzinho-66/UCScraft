# UCScraft

## Instruções

> WASD: Voar pelo cenário
>
> Shift: Correr
>
> 1~4: Escolher um bloco
>
> Botão esquerdo do mouse: Colocar um bloco na posição do cursor
>
> Botão direito do mouse: Quebrar o bloco na posição do cursor
